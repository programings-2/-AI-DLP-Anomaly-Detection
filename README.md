# 🔐 AI-Driven Data Loss Prevention (DLP)

## 📌 Overview
This project simulates a Data Loss Prevention system using Artificial Intelligence to detect suspicious data transfer activities.

## ⚙️ Features
- Simulation of user activity dataset
- Anomaly detection using Isolation Forest
- Interactive dashboard using Streamlit
- Visualization of suspicious behavior
- Export results to CSV

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Streamlit

## ▶️ How to Run

1. Install dependencies:
pip install streamlit pandas matplotlib scikit-learn

2. Run the dashboard:
streamlit run app.py

## 📊 Output
- Detection of anomalies
- Graph visualization
- List of suspicious activities
